# App Store Release Notes - Version 0.9.5

## What's New in Version 0.9.5

**Fixed Charity Display Bug** - Resolved an issue where the wrong charity would sometimes appear on leaderboard results. Each workout now correctly displays the charity you selected when you published it.

**Improved Donation Experience** - Made it easier to support your favorite Bitcoin charities! The donation screen now features:
- Quick amount selection with preset buttons (1000, 2100, 5000, or 10000 sats)
- Custom amount input for flexible donations
- "Set as default" option to remember your preferred donation amount
- Streamlined interface focused on charity donations

This update focuses on fixing bugs and improving the user experience when supporting Bitcoin charities through your fitness achievements.

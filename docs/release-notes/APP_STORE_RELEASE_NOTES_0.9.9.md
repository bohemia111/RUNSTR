# App Store Release Notes - Version 0.9.9

## What's New in Version 0.9.9

**Enhanced Activity Tracking**
Completely redesigned GPS tracking system for better accuracy and reliability. Smart GPS watchdog detects and recovers from tracking failures automatically. Battery-adaptive GPS modes optimize power usage during workouts.

**Running Personal Records**
Now displays your best times for 5K, 10K, Half Marathon, and Marathon right on the activity card. Track your progress toward new PRs with each run.

**Walking Improvements**
New "Best Walk" display shows your walk with the most steps tracked. Clearer step counting with platform-specific labels.

**Android Fixes**
- Fixed step double-counting issue in Health Connect
- Corrected exercise type mappings for 20+ activities
- Improved background tracking reliability with zombie interval detection

**Reliability Improvements**
- Fixed iOS 30-minute tracking crash issue with intelligent write batching
- Better elevation tracking during workouts
- Unified tracking service architecture for consistency
- Session persistence across app restarts

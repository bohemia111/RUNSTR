# Screens Directory

Main application screens for the RUNSTR mobile app's user interface.

## Subdirectories

- **activity/** - Activity tracking screens (running, cycling, walking, meditation, strength, diet)
- **season2/** - Season II event screens
- **satlantis/** - Satlantis event screens
- **events/** - Event detail screens

## Main Tab Screens

- **ProfileScreen.tsx** - Profile tab: workout tracking, history, settings
- **TeamsScreen.tsx** - Teams tab: charity selection and display
- **RewardsScreen.tsx** - Rewards tab: earnings, Impact Level, donation splits

## Authentication

- **LoginScreen.tsx** - Nostr nsec authentication
- **ProfileImportScreen.tsx** - Initial profile setup

## Workout & Fitness

- **WorkoutHistoryScreen.tsx** - Unified workout timeline (HealthKit + Nostr)
- **AdvancedAnalyticsScreen.tsx** - Fitness analytics and Coach Runstr
- **HealthProfileScreen.tsx** - Health metrics and body composition

## Events & Competitions

- **EventsScreen.tsx** - Browse available events
- **CompetitionsListScreen.tsx** - User's competitions
- **LeaderboardsScreen.tsx** - Event leaderboards

## Settings & Support

- **SettingsScreen.tsx** - App settings
- **WalletScreen.tsx** - Lightning address configuration
- **HelpSupportScreen.tsx** - Help and support
